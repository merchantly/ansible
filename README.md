# db-server-ansible

## Подготовка

1. Прежде чем запускать ansible, установите его локально.

> make galaxy

2. Установите на удаленный сервер.

> make install

Подробнее: https://www.digitalocean.com/community/tutorials/how-to-install-and-configure-ansible-on-ubuntu-14-04

3. Проверьте настройку:

> make ping
